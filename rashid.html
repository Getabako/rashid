小イーグルスパイク後

  • 画面端
    • しゃがみ中P → しゃがみ中P → 中P → 中アラビアンサイクロン → 弱K → 弱イーグルスパイク
    • → ガードされた場合：歩き投げ
    • しゃがみ中P → 中P → 中K → 大アラビアンサイクロン

  • 中K → 前ステップ → 投げ

  • → 端に到達した場合：弱K → 強P
  • → 端に到達してない場合：中K
